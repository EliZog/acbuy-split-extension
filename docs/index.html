Privacy Policy

Split Parcel Extension

Single Purpose

The single purpose of Split Parcel is to provide a visual interface for users on the ACbuy website to organize, calculate, and split the costs of shipping parcels among multiple individuals based on item assignments.

Permission Justification

Split Parcel requests only the minimum necessary permissions to fulfill its single purpose.

Host Permissions

  • https://www.acbuy.com/*
  • https://acbuy.com/*

Justification: These permissions are required to inject the Split Parcel interface directly into the ACbuy dashboard, discover existing parcel IDs, and fetch the necessary parcel and order details required to perform the cost-splitting calculations.

Remote Code

Split Parcel does not use any remote code. All logic, including scraping and UI management, is contained within the extension package.

Data Usage

By publishing this item, the developer certifies that these disclosures reflect the most up-to-date content of the privacy policy.

User Data Collection

The extension collects the following types of data purely for local processing within the user's browser session:

  • Website Content: The extension scrapes item names, thumbnails, prices, and parcel IDs from the ACbuy website (specifically from the user's own "My Package" and "Order History" pages) to populate the splitting interface.

The extension does NOT collect or store:

  • Personally identifiable information (name, address, email, etc.)
  • Health information
  • Financial and payment information (credit card numbers, bank details, etc.)
  • Authentication information (passwords, credentials)
  • Personal communications
  • Location data
  • Web history (beyond the ACbuy domain required for operation)
  • User activity tracking (keystroke logging, network monitoring, etc.)

  2. I do not use or transfer user data for purposes that are unrelated to my item's single purpose.
